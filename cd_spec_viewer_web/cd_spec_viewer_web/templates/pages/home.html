CD Spec Viewer

Welcome to Dr. Sherrer's CD Spec Viewer

Description: The Circular Dichroism Spectrometer (CD Spec) application is an online database created for research and education purposes. This application shall be utilized by both the instructors and students to monitor spectral changes in structure between protein-DNA interactions.

Disclosures: A condition for use of the data in this website is that any publication or presentation using any data from the CD Spec Viewer database must cite both the original reference for any externally listed data (noted in the individual records) plus the CD Spec Viewer reference below:

Shanen M. Sherrer (2021). CD Spec Viewer [Database]. Circular Dichroism Spectral Data Access from CD spectrometers. Retrieved from https://cdspeccollab.smcm.edu.

The CD Spec Viewer was developed by Dr.Shanen M. Sherrer and collaborators with financial support from the National Science Foundation (NSF MRI grant#1919581). Copyright for the design and implementation of this site are retained by the Dr. Sherrer and St. Mary's College of Maryland.
